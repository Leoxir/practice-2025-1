Проект включает создание 3D-тренажёра для симуляции пусконаладочных испытаний оборудования автоматизированных систем (АС), направленного на повышение эффективности обучения студентов I курса.

О проекте

Проект «Создание 3D-тренажёра для проведения пусконаладочных испытаний на оборудовании АС» направлен на развитие навыков студентов в области автоматизированных систем.

Цели проекта

  • Разработка интерактивной платформы для виртуальных испытаний оборудования (датчики, контроллеры, интерфейсы).
  • Автоматизация проверки сценариев пусконаладки, таких как калибровка сенсоров и диагностика неисправностей.
  • Интеграция элементов симуляции для обучения без использования реального оборудования.

Используемые технологии

  • Unity/Blender: для 3D-моделирования оборудования АС.
  • C#: для скриптов симуляции процессов пусконаладки.
  • Git: для управления версиями и хранения исходников.
